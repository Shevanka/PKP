# Praktikum Konsep Pemrograman
Repository ini berisi rangkaian tugas, praktikum, dan dokumentasi untuk mata kuliah Praktikum Konsep Pemrograman di Politeknik Elektronika Negeri Surabaya.

## Isi Repository
- Praktikum mingguan (Folder Minggu-01 sampai Minggu-14)
- Laporan resmi sesuai format laboratorium
- Catatan pribadi dan dokumen pendukung (folder /notes dan /docs)
- Final Project (Tugas Akhir)
